## Report Format

All reports follow a consistent structure:
1. Executive Summary
2. Incident Overview
3. Detection Source
4. Timeline of Events
5. Indicators Observed
6. Investigation Notes
7. Containment Actions
8. Remediation Recommendations
9. Lessons Learned
10. Evidence / Screenshots
